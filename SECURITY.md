# Security Policy

## Supported Scope

This repository contains intentionally vulnerable training/demo applications and security pipelines.

- The code under `php/`, `python/`, and `reactjs/` may include known vulnerabilities by design.
- Only pipeline, infrastructure, and repository-integrity issues are treated as reportable security issues.

## Reporting a Vulnerability

If you find a security issue that is **not intentionally vulnerable training content**, report it privately:

- Open a GitHub Security Advisory draft for this repository, or
- Contact the repository owner directly through GitHub profile contact options.

Please include:

- Affected path(s) and branch/commit
- Reproduction steps
- Impact assessment
- Suggested fix (if available)

## Response Expectations

- Initial triage target: within 7 days
- Valid issues are prioritized by severity and exploitability
- Fixes may be shipped directly to `main` or through pull requests

## Disclosure

Please avoid public disclosure until maintainers confirm a fix or mitigation.
